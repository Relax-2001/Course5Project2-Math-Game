# 🧠 Math Quiz Game (C++ Console Application)

## 📌 Description

A simple **console-based math quiz game** built using C++.
The game generates random math questions and lets the player answer them while tracking performance.

## ✨ Features

* Choose number of questions (1 – 10)
* Select difficulty level:

  * Easy
  * Mid
  * Hard
  * Mixed Level
* Select operation type:

  * Addition
  * Subtraction
  * Multiplication
  * Division
  * Mixed Operations
* Tracks:

  * Correct answers
  * Wrong answers
  * Pass/Fail result
* Random question generation

## 🛠️ How to Run

1. Clone the repository from GitHub.

2. Compile the project:

```bash
g++ main.cpp -o quiz
```

3. Run the program:

```bash
./quiz
```

## 🎮 Game Instructions

* Enter number of questions (1–10).
* Choose difficulty level.
* Choose operation type.
* Answer each question when prompted.

## 👨‍💻 Technical Details

* Uses **structs** to organize quiz data.
* Uses **enums** for game states and options.
* Random number generation for question creation.
* Modular function-based architecture.
